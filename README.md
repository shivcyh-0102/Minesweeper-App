## 🎮 Gameplay

| Action | How |
|---|---|
| Reveal a cell | Tap |
| Place / Remove flag 🚩 | Double-tap |
| Restart game | Tap 🔄 button |

---

## ✨ Features

- 9×9 grid with 10 mines
- Auto-reveal empty surrounding cells (flood fill)
- Mine counter
- Win & Loss detection
- Restart anytime

---

## 🛠️ Built With

- [.NET MAUI](https://learn.microsoft.com/en-us/dotnet/maui/) — Cross-platform UI framework
- **C#** — Game logic
- **XAML** — UI layout
- **MVVM** Architecture — Clean separation of logic and UI
